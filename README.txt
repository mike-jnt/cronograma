## Recomendado para evitar el 404 en celular
1. Activa GitHub Pages desde la rama correcta.
2. Espera a que la URL publique bien.
3. Abre la URL del sitio en el teléfono.
4. Si ya tenías una instalación vieja, **elimínala** del teléfono.
5. Borra los datos del sitio en el navegador.
6. Vuelve a entrar y vuelve a instalar la app.
